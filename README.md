🏋️ Fitness Analytics Dashboard
A data-driven dashboard designed for a local gym to track and analyze member activity,their status and overall fitness engagement. Built to help gym owners, trainers, and members gain insights into performance, retention, and wellness trends.

📌 Features
Finances - Brief summary of the revenue, expenses and profit of the gym for the year.

Calorie Calculation - Records of the optimum BMI & BMR for healthy body.

Membership Status- Track of the age group of members, gender and their membership status

Trainer & Members - Record of the active members and trainers assigned to them.

Interactive Filters – Filter by time period, membership type, or fitness program.

⚒️ Tech Stack
#🛠️ Figma- For interactive backgrounds used in the project.

#📊 Power BI Desktop- Main data visualisation platform used for dashboard creation.

#📁 Power Query- Data transformation and cleaning for reshaping the data.

#🧠 DAX(Data Analysis Expresssions)- For calculating different measures, dynamic visuals and conditional formatting.

#📄 Data Modelling- For establishing relationships between tables.

#📁 File Format- .pbix for development and .png for dashboard previews.

🗄️Data Source
The data for this dashboard was collected from:

Member registration and attendance logs exported from the gym’s management system.
Trainer performance and class feedback forms.
✨ Project Highlights
📈 End-to-End Analytics – Covers membership, trainers, and fitness progress thedashboards.

💡 Member Retention Focus – Tracks churn, engagement trends, and high-risk inactive members.

⭐ Trainer & Class Evaluation – Analyzes trainer performance, trainer popularity, and membership level.

📊 Visual Storytelling – Easy-to-understand charts, KPIs, and dashboards for non-technical users.
